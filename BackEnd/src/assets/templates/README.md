# Templates de Relatórios — ProntuSMART

## Marca d'água oficial

Coloque o arquivo **`marca_dagua.pdf`** nesta pasta. Ele será usado como
fundo (overlay) em todos os relatórios PADRÃO gerados pelo sistema.

- **Nome esperado:** `marca_dagua.pdf`
- **Formato:** PDF de 1 página, A4 (210×297 mm)
- **Conteúdo recomendado:** logos institucionais (UCB, Curso de Fisioterapia)
  no topo e endereço/contato no rodapé. Toda a área central do documento
  deve ficar livre para o conteúdo do relatório.

> Se o arquivo não for encontrado, o sistema gera o PDF apenas com o
> cabeçalho textual padrão (sem prejuízo funcional, mas sem identidade
> visual da clínica-escola).
